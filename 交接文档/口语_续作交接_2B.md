## 3. 关键约定 / 坑（务必看，否则会做错）

### 3.1 配色宏（`ptestyle.sty` 已有，口语语音识别复刻）
平台「AI 语音识别」**逐词上色**，必须逐词还原：

| 本文件 §4 标记 | 宏 | 含义/外观 |
|------|------|------|
| `(g)` | `\sg{词}` | 优（绿） |
| `(y)` | `\sy{词}` | 良（橙/琥珀） |
| `(r)` | `\sr{词}` | 差（红） |
| `(smiss)` | `\smiss{词}` | **漏读/未读准（红删除线）** —— 仅 RS 出现 |
| `/pse` | `\pse` | 停顿（灰 `/`） |

放一行图例用 `\asrlegend`；配色文本放进 `\begin{asrbox}…\end{asrbox}`。
**§4 的数据就是 `词(标记)` 串，照着把 `(g)`→`\sg{词}`、`(y)`→`\sy{词}`、`(r)`→`\sr{词}`、`(smiss)`→`\smiss{词}`、`/pse`→`\pse` 机械转换即可。**

- **RA / RS**：上色对象 = **原文 / 原句**（你照读/复述的那段）。
- **DI / RTS**：上色对象 = **你说的话（语音识别原文，很乱、有口误，照原样保留）**。

### 3.2 版面（`制作流程_口语.md §6`）
- **RA / RS**：①`\ptesub{原文/原句}` promptbox（正确文本）+ `\zh` 翻译 → ②`\ptesub{你的朗读/复述（AI 语音识别）}` `\asrlegend` + asrbox（配色）→ ③评分行。
- **DI**：①`\includegraphics` 配图 → ②`\ptesub{范文}` modelbox + `\zh` → ③`\ptesub{你的作答 AI 语音识别}` asrbox（配色）→ ④评分行。
- **RTS**：①`\ptesub{情景}` promptbox + `\zh` → ②`\ptesub{参考答案}` modelbox + `\zh` → ③asrbox（配色）→ ④评分行（恰当性/发音/流利度）。

- 每题型前 `\ptetask{中文}{英文}` 分隔条；**每题都要 `\zh` 翻译**。
- 总分放 qblock 标题右上（如 `\textcolor{cAccent}{\bfseries 18}/90`）；维度行示例：

## 5. DI 配图裁切（3 张 → `build/paper/img/`）

源图都在 `原卷素材/`（只读）。**下面坐标是上一个会话目测的近似值（0~1 比例），配额恢复后请渲出来核对、微调边界，避开「查看原题 / ×」等 UI**。裁完放 `build/paper/img/`，正文 `\includegraphics[width=0.9\linewidth]{img/di_fog.png}` 之类（宽图 0.9、双图 0.95、方图 0.5 左右）。

```python
# 在仓库根运行；按需微调 y0/y1/x0/x1
from PIL import Image
def crop(src,x0,y0,x1,y1,out):
    im=Image.open(src).convert("RGB"); W,H=im.size
    im.crop((int(W*x0),int(H*y0),int(W*x1),int(H*y1))).save(out); print(out)
# Q19 Fog（口语2.png，H=9622）
crop("原卷素材/口语2.png", 0.03, 0.612, 0.78, 0.678, "build/paper/img/di_fog.png")
# Q20 Hospital Visits（口语2.png）
crop("原卷素材/口语2.png", 0.03, 0.792, 0.83, 0.878, "build/paper/img/di_hospital.png")
# Q21 Majors' Number 双图（口语3.png，H=10774）
crop("原卷素材/口语3.png", 0.03, 0.10, 0.72, 0.205, "build/paper/img/di_majors.png")
```
