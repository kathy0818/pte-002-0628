# 制作流程 · 写作分册（Writing）

> 配合总册 `制作流程.md` 使用。本册讲**写作部分**怎么做：题型、要素、批改约定、补标、翻译、逐题 LaTeX 结构，附 2B 实例。
> 工作文件：`build/paper/sec_writing_body.tex`（正文）、`preview_writing.tex`（预览）、`cover.tex`（封面）。

---

## 1. 输入与题型

- **总览/题目卷**：`原卷素材/写作1.png`（写作整页长截图：每题题干/原文、**WE 范文**、你的作答、得分、用时）。
- **详解卷** ⭐：`原卷素材/作文详解.docx`（内嵌 **4 张「AI 评分」弹窗截图** = 每题逐项评分 + 彩色批改 + AI 中文建议）。
  - ⚠️ 本套（2B）详解是 **docx（内嵌截图）**，不是 1B 的 PDF。docx 没有文字层，先导出内嵌图再肉眼誊录：
    `unzip -o -j 原卷素材/作文详解.docx 'word/media/*' -d /tmp/w && ls /tmp/w`（得 image1~4.png，逐张 Read 看）。

PTE Core 写作两种题型：

| 题型 | 满分 | 每题要素 |
|------|------|----------|
| **SWT 概括写作** Summarize Written Text | 8 | 原文 Source Text → 你的作答(含批改) → 评分(4 维) → AI 建议 |
| **WE 邮件写作** Write Email | 15 | 题干 Task Prompt(含 3 个 themes) → **范文 Model Answer** → 你的作答(含批改) → 评分(7 维) → AI 建议 |

> 题量以实际为准（1B、2B 均为 SWT×2 + WE×2）。SWT 没有平台范文，WE 有。

**评分维度**（照抄平台「评分明细」表，每行：维度 + 得分 + AI 建议）：
- **SWT**：Content 内容 /2、Form 格式 /2、Grammar 语法 /2、Vocabulary 词汇 /2。
- **WE**：Content /3、Form /2、Email Conventions 邮件格式 /2、Organization 结构 /2、Vocabulary /2、Grammar /2、Spelling 拼写 /2。

## 2. 批改约定（核心）

平台用颜色标错误，本卷用宏复刻（都在 `ptestyle.sty`）：

| 平台标注 | 本卷宏 | 用法 |
|----------|--------|------|
| 红色改写 / 拼写错 / 缺空格 | `\fix{原词}{正确}` | `\fix{transfor}{transform}`、`\fix{regularsports}{regular sports}` |
| 该删的词 | `\del{词}` | `\del{more}` |
| 该插入的词 | `\ins{词}` | 较少用 |
| **平台漏标、本卷补标** ⭐ | `\add{原}{改正}` | `\add{replay}{reply}`（橙波浪线 + “补”） |

**⭐ 补标（增值点）**：仔细核对原图，平台**漏标**的明显错误，用 `\add` 橙色“补”标出，并在「本题得分」行用 `\textcolor{cAdd}{补标：…}` 说明。这是本卷比平台更细的地方，必须做。
- 1B 实例：Q3 平台漏标 `replay→reply`；Q4 漏标 `innovative` 前缺动词（应 introduce）。
- 2B 实例：Q3、Q4 平台都漏标了 `Looking forwards → Looking forward`（固定搭配 look forward to），本卷用 `\add` 补标。

> 注意：`\add` 第二参数可写中文说明，如 `\add{innovative}{前缺动词，如 introduce}`。

## 3. 翻译（每题都加）

用 `\zh{中文}` 给以下英文加翻译，放在对应框正下方：
- **SWT**：原文 Source Text 的翻译。
- **WE**：题干 Task Prompt 的翻译 + 范文 Model Answer 的翻译。

（你的作答含彩色批改，不翻译。）

---

## 4. 逐题 LaTeX 结构

每题一个 `qblock`。模板（SWT）：

```latex
\begin{qblock}{Q1\quad SWT \#57\ \ Metaverse}{7.5 / 8\quad·\quad 用时 09:06}
\ptesub{原文 Source Text}
\begin{promptbox} ……原文照抄…… \end{promptbox}
\zh{……原文中文翻译……}

\ptesub{你的作答 Your Response}
\begin{youbox} ……你的作答，错处用 \fix/\del/\add 标…… \end{youbox}

\ptesub{评分明细 Score Breakdown}
{\small\scorehead
\begin{tabularx}{\linewidth}{l c Y}
\rowcolor{cHead}\textcolor{white}{\bfseries 维度} & \textcolor{white}{\bfseries 得分} & \textcolor{white}{\bfseries AI 建议}\\
Content 内容 & \sfull{2 / 2} & 内容很棒\\
……每个维度一行，满分用 \sfull、部分 \spart、零分 \szero……
\end{tabularx}}
\smallskip
\textbf{本题得分：}\ {\color{cAccent}\bfseries 7.5 / 8}\quad{\footnotesize\color{gray}（错误：\fix{transfor}{transform}）}
\end{qblock}
```

WE 题在「原文」位置改用 `\ptesub{题干 Task Prompt}`（三个 themes 用 `itemize`），并在你的作答前多一段：

```latex
\ptesub{范文 Model Answer（平台参考）}
\begin{modelbox} ……范文…… \end{modelbox}
\zh{……范文翻译……}
\ptesub{你的作答 Your Response（含批改）}
\begin{youbox} …… \end{youbox}
```

> 表头列宽用文件顶部已定义的 `\newcolumntype{Y}{>{\RaggedRight\arraybackslash}X}`。直接照抄现有 `sec_writing_body.tex` 最稳。

**封面 `cover.tex`**：写作分数填进四项条形图（条长 = 得分/90），并改提交时间、总分。

## 5. 2B 实例（本套已完成 —— 写作 `000`）

本套写作总分 **71 / 90**（四题里 SWT 两题满分，WE 两题主要丢在语法）：

| 题 | 题型 | 主题 | 得分 | 关键错误（含补标） |
|----|------|------|------|--------------------|
| Q1 | SWT \#60 | Elizabeth Blackwell（医学先驱）| 8/8 | 无（四维满分）|
| Q2 | SWT \#69 | Beta Testing（软件测试）| 8/8 | 无（四维满分）|
| Q3 | WE \#10 | Healthier Work Environment（职场健康）| 12.9/15 | `supervisor→Supervisor` 大写、`I'm writing` 缺空格、`, company→, the companies`、`food` 后缺逗号（插入）、`more` 删；**补标 Looking forwards→forward**。语法 0.4/2 |
| Q4 | WE \#11 | Remote Team Collaboration（远程协作）| 12.5/15 | `leader→Leader` 大写、`I'm writing` 缺空格、`suggesting→suggest`、`improves→improve`×2、`adopting→adopt`、`scheduling→schedule`；**补标 Looking forwards→forward**。语法 0/2 |

> 复盘要点：两封邮件**内容/结构/格式全满**，**语法（动词形式）几乎全扣**——`can / to` 后该用动词原形，作答里多次误用 `-ing / -s`。这是 WE 提分的最快抓手。

成品见 `整理卷/写作/`：`000`（首版，四题齐全、含翻译与补标）；最终成品 PDF 复制在 `整理卷/PTE模考2B_写作_整理卷.pdf`。**下一套从 `000` 重新起编号。**

> 本套两道 SWT 原文、两道 WE 题干/范文均完整、无分页截断；逐字以 `原卷素材/写作1.png` 为准誊录。

## 6. 出版本 & 自检清单

写作无图 → 纯文字版本：

```bash
cd build/paper && xelatex preview_writing.tex && xelatex preview_writing.tex   # 预览核对
python3 build/flatten.py writing 写作 <编号> <序号>     # → 整理卷/写作/<序号>_…_写作_整理卷.tex
cd 整理卷/写作 && xelatex <序号>_PTE模考<编号>_写作_整理卷.tex && xelatex … && rm -f *.aux *.log *.out
# 最终版：cp <序号>_PTE模考<编号>_写作_整理卷.pdf ../PTE模考<编号>_写作_整理卷.pdf   （整理卷/ 根只放最终成品）
```

**自检（逐题对原图）**：
- [ ] 原文 / 题干 / 范文 一字不差，且各有 `\zh` 翻译。
- [ ] 你的作答原样誊录，错处颜色/类型与平台一致（`\fix/\del`）。
- [ ] 评分明细每个维度的分数、AI 建议原文照抄。
- [ ] 平台漏标的错误已用 `\add` 补标，并在得分行说明。
- [ ] 「本题得分」与平台一致（平台显示值与精确值不符时，标注两者，如 7.25 与“平台显示 7.3”）。
- [ ] 截断/缺失处有红字说明。
