  —— 复盘页里 **MCM-L / FIB-L / MCS-L / SMW / HIW 都直接给出录音 Transcript**（WFD 给正确句子），非常省事。

- **⭐ SST 评分明细**：`原卷素材/听力0.png`（= `听力详解1.pdf` 的清晰版）—— SST 的 **Content/Form/Grammar/Spelling/Vocabulary 五项打分 + AI 建议 + 你概述的错词标注**。
- **SST 音频原文**：复盘页**不含**，需在平台点该题「查看原题」复制（1B 的 #21 原文由考生补来）。

|------|------|----------|

## 2. ⭐ 听力专用宏（`ptestyle.sty`）

| 宏 | 用于 | 外观 |
|----|------|------|
| `\rok{词}` / `\rno{你的}{正确}` | FIB-L 填空对错 | 同阅读（绿✓ / 红✗+答案）|
| `\hiw{屏幕错词}{实际读音}` | **HIW** | 屏幕错词**红下划线** + `(听:实际读音)` |
| `\lhit{词}` | **WFD** 命中 | 绿 |
| `\lmiss{词}` | **WFD** 多写/听错（**1B 配色**）| 红删除线 |
| `{\color{gray}(词)}` | **WFD** 漏写（**1B 配色**）| 灰括号 |
| **`\womit{词}`** | **WFD** 漏写（**本套 2B 配色**）| **红色括号** `(词)` |
| **`\wbad{词}`** | **WFD** 多写/听错（**本套 2B 配色**）| **灰删除线** |
| **`\wfdlegend`** | WFD 一行图例 | 命中 / (漏写) / 多写听错 |

> **WFD 两套配色别搞混**：1B 用 `\lhit`(绿) + 灰括号(漏写) + `\lmiss`(红删，多写)；**2B 平台把颜色对调了**，用 `\lhit`(绿命中) + `\womit`(红括号，漏写) + `\wbad`(灰删，多写/听错)。两套宏都在 `ptestyle.sty` 里（只增不改），做新一套时**先放大看平台到底哪种颜色对应漏写/多写，再选宏**。

**HIW 原理**：屏幕上显示的文字与录音不符，要**点出与录音不同的词**。平台给：`答案: Pronounced words: …`（= 实际读音/正确词）。把屏幕错词逐个 `\hiw{屏幕}{读音}` 标进原文，再补一行汇总「应点的 N 个错词 / 你点了哪几个」。

**WFD**：平台逐词批改——**绿=命中**、**(灰)=漏写**、**红删=多写或听错**。例：
`\lhit{Different} {\color{gray}(factors affect the freezing)} \lhit{water} \lmiss{often} \lhit{time} {\color{gray}(of)} \lmiss{times}`

## 3. ⭐ PDF × 长截图 互补（听力的坑）

- 听力 PDF 清晰，但 **① 分页缝**会吞掉行（如 MCM-L 的 `你的答案` 行）、**② 右下角悬浮按钮**会盖住内容。
- **长截图 `听力1/2.png` 连续无缝、无悬浮遮挡** → 用 `build/pngcrop.py` 按比例裁来补：
  ```bash
  python3 build/pngcrop.py 原卷素材/听力1.png <y0> <y1> <名字>
  ```
- 实操：题目地图用 PDF（`pdftoppm` 整本渲图 + `zoom.py` 读小字）；被缝/按钮吞掉的 `你的答案`、`评分` 行去长截图补。
- SST 评分明细只在 `听力0.png` 里，单独读。

## 4. 录音原文 & 翻译（每题都加）

- 录音原文（Transcript）直接照抄复盘页（**注意是听力口语稿，可能有口误/不规范，照原样**）。SST 原文需「查看原题」另取。
- `\zh{中文}` 给**每段录音原文 / 每个 WFD 句子**加翻译。
- SST 还要保留**你的概述**（含 AI 标注的待改词，用 `{\color{cAccent}\uline{…}}` 标）+ 五项评分表。

---

## 5. 逐题 LaTeX 结构

- **SST**：`\ptesub{录音原文}` promptbox + `\zh` →`\ptesub{你的概述}` promptbox →`\ptesub{AI 评分明细}` notebox 内 `tabular`（Content/Form/Grammar/Spelling/Vocabulary 各一行 + 得分 + 建议）。
- **MCM-L / MCS-L**：`\ptesub{录音原文}` + `\zh` + `\ptesub{题目}` + `\ptesub{选项}`（正确绿粗）+ 答案行。
- **FIB-L**：`\ptesub{录音原文（含填空作答）}` promptbox（空用 `\rok/\rno`）+ `\zh` + 一行说明你的作答（打字题无选项）。
- **SMW**：录音原文结尾用 `\rule{1.2cm}{0.4pt}\ \textit{[beep]}` 表示被替换的词 + `\zh` + 选项 + 答案行。
- **HIW**：录音原文（`\hiw` 标错词）+ `\zh` + 一行「应点的 N 个错词 / 你点了…」。
- **WFD**：`\ptesub{正确句子}` promptbox + `\zh` + `\ptesub{你的作答（绿命中/灰漏写/红多写）}` + 逐词批改串。
- 标题右上放评分：`… · 评分 \textcolor{cAccent}{\bfseries 7 / 10}`。

## 7. 出版本 & 自检清单

听力无图 → 纯文字版本：

```bash
cd build/paper && xelatex preview_listening.tex && xelatex preview_listening.tex   # 预览核对
python3 build/flatten.py listening 听力 <编号> <序号>    # → 整理卷/听力/<序号>_…_听力_整理卷.tex
cd 整理卷/听力 && xelatex <序号>_PTE模考<编号>_听力_整理卷.tex && xelatex … && rm -f *.aux *.log *.out
# 最终版：把该 pdf 复制到 整理卷/ 根并改成干净名 PTE模考<编号>_听力_整理卷.pdf（见总册 §4）
```

**自检（逐题对原图）**：
- [ ] 录音原文一字不差（含口误），每段有 `\zh` 翻译。
- [ ] FIB-L 空处 ✓/✗ 对；MCM/MCS/SMW 选项正确项绿粗、答案行对错配色对。
- [ ] HIW：错词 `\hiw` 标全、`(听:…)` 对，汇总行「应点 N 个 / 你点了…」与平台一致。
- [ ] WFD：正确句子准确，逐词批改（绿/灰/红）与平台一致，得分对。
- [ ] SST：录音原文（查看原题取）、你的概述、五项评分明细齐全。
- [ ] PDF 被缝/悬浮按钮吞掉处，已用长截图 `听力1/2.png` 补全。
