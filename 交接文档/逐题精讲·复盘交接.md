## 2. 看懂「对错 / 配色」宏（判断每题对错的钥匙）

`.tex` 里用这些宏标注我的作答对错和平台配色。**照此判断**：

**阅读 / 听力（选择 · 填空 · 找错 · 听写）**

| 宏 | 含义 | 对/错 |
|----|------|------|
| `\rok{词}` | 我**选对/填对** | ✅ 对 |
| `\rno{我的}{正确}` | 我**选错/填错**：红删我的错答 + 括注正确答案 | ❌ 错 |
| `\rblank{答案}` | 直接给出的正确答案（无作答对照时）| 参考 |
| `\hiw{屏幕词}{实际读音}` | HIW：屏幕上与录音**不符的词**（本该点出）；题末汇总「应点 N 个 / 你点中 M 个」| 看汇总判断点中没 |
| `\lhit{词}` | WFD：**命中**（写对了）| ✅ |
| `\womit{词}` | WFD：**漏写**（答案有、我没写）| ❌ 漏 |
| `\wbad{词}` | WFD：**多写/听错**（我写了但错）| ❌ 多/错 |

**写作（批改）**

| 宏 | 含义 | 对/错 |
|----|------|------|
| `\fix{原}{正}` | 我的**拼写/语法/用词错** → 红删原词 + 绿色正确 | ❌ |
| `\del{词}` | 多余、**该删** | ❌ |
| `\ins{词}` | **漏了、该插入** | ❌ 漏 |
| `\add{原}{正}` | 平台漏标、**本卷补标**的错 | ❌ |

**口语（逐词发音质量 + 评分）**

| 宏 | 含义 | 提示 |
|----|------|------|
| `\sg{词}` / `\sy{词}` / `\sr{词}` | 该词发音 **优 / 良 / 差** | `\sr` 红 = 发音弱点 |
| `\smiss{词}` | **漏读 / 未读准**（红删除线）| ❌ 漏 |
| `\pse` | 停顿（灰 `/`）| 流利度参考 |

**分数**（所有 part 通用）：`\sfull{x/y}`=满分(绿)、`\spart{x/y}`=部分(橙)、`\szero{x/y}`=零/低分(红)。
每题得分在 **qblock 标题右上**、**「评分：」行**、和题末灰字（用时/简评）里。

## 3. 逐题讲解模板（每道题都按这 7 块输出）

> 对的题第 ⑦ 块可精简，但要指出「为什么没拿满 / 还能更好」。错题第 ⑦ 块是**重头戏**，必须写足。

1. **题号 · 题型 · 标题 · 得分** —— 例：`Q6 MCS-L #127 Autism Diagnoses —— 0/1`。
2. **题目 / 材料** —— 1–2 句说清这题考什么、给了什么（原文/录音稿/图/情景），复杂的给原文要点 + 中文。
3. **正确答案 / 范文要点** —— 标准答案是什么；范文好在哪（结构/句式/关键信息）。
4. **精讲——为什么对** —— 考点、判断依据、**原文定位句**、同义替换、逻辑/语法点。**对的题也要讲清「凭什么是它，干扰项错在哪」**，提炼能迁移的通法。
5. **解题思路 / 套路** —— 这一类题的标准打法、技巧、时间分配、排除法。
6. **我的作答 & 判定** —— 我答了什么、对/部分对/错，**逐处**指出（依据第 2 节的宏）。
7. **★ 若错或部分对——错因 + 防错（错题重头戏）**：
   - **错因归类**：知识点不会 / 没看出同义替换 / 粗心看漏 / 听不清·拼写错 / 逻辑连接判断错 / 时间不够 / 发音·流利度弱 …（指明属于哪一类）。
   - **防错清单**：下次遇到这类题，**具体怎么做才不再错**（可操作的检查动作，1–3 条）。
   - **同类题通法 + 一句话记忆点**。

## 4. 各题型该讲的重点（讲解时的考点库）

**听力**
- **SST 概述口语**：抓主旨 + 关键信息，50–70 词；评分五维 Content/Form/Grammar/Spelling/Vocabulary——讲我哪一维弱（本套 Q1 = 8.9/10，弱在 Grammar、Vocabulary）。
- **MCM-L / MCS-L**：同义替换 + 排除干扰项；**MCM 多选错选/漏选都不得分**，要稳。
- **FIB-L 听力填空（打字）**：边听边打、拼写关；讲我把什么词听成了什么（本套多为听错/拼写：strenthen/diliver/abrot…）。
- **SMW 选词**：抓句末语义走向选缺词。
- **HIW 找错词**：屏幕词 vs 实际读音，**宁可少点别乱点**（点错相当于扣分）；讲漏点了哪些、为什么没听出。
- **WFD 听写整句**：性价比最高、争取拿满；逐词记，分析漏写/多写/听错；讲记短句的方法（意群、缩写）。

**阅读**
- **FIB（下拉选词）/ FIBD&D（拖词填空，即 FIB(R)）**：搭配(collocation)、词性、上下文逻辑；拖词题注意**共享词库 + 排除法**。
- **MCM-R / MCS-R**：定位 + 同义替换 + 排除干扰项。
- **RO 段落重排**：找**首句**（独立、引入话题），再用**代词 / 连接词 / 时间·因果线索**串顺序。

**口语**（本套内容/恰当性中上，**发音、流利度普遍偏低是主失分点**）
- **RA 朗读**：发音、连读、重音、断句、流利度 → 讲**影子跟读**、按意群断句、`\sr`/`\pse` 暴露的具体弱词。
- **RS 复述**：先抓全句再复述，漏词扣内容分（看 `\smiss`）。
- **DI 描述图**：用固定模板（开头一句 + 主体数据/对比 + 结尾趋势/总结），**结合 `img/` 配图**讲怎么把图的关键信息说全（本套 Q19 几乎没描述出来 → 重点讲 DI 框架）。
- **RTS 情景应答**：恰当性模板（招呼 + 回应对方 + 你的请求/建议 + 收尾），讲怎么把话说得体。
- **ASQ 简答**：考词汇量与反应，常考定义词（本套只 Q25 Elephant 答对，其余 Contemporary/Ceiling/Mystic/Parade/Section 没答出 → 讲这些词）。

**写作**
- **SWT 概括写作**：**用一个单句**概括全文（主句 + 从句），抓主旨、不超字数、语法拼写零失误。
- **WE 邮件**：结构（称呼 + 目的 + 展开 2–3 点 + 结尾）、语气（正式/半正式）、连接词；讲 `\fix/\add` 标出的语法用词错。
