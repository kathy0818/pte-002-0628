## 2. ⭐ 配色复刻（最关键，别偷懒）

详解卷的「AI 语音识别」是**逐词上色**的，**必须逐词还原**（曾因只写文字摘要被指出）。图例：
**绿=优 / 橙=良 / 红=差 / `/`=停顿 / 红删除线=漏读·未读准**。

用 `ptestyle.sty` 的宏：

```latex
\asrlegend                       % 先放一行图例
\begin{asrbox}
\sg{绿词} \sy{橙词} \sr{红词} \pse \smiss{漏读词} ……
\end{asrbox}
```

- `\sg`=优(绿)、`\sy`=良(橙)、`\sr`=差(红)、`\smiss`=漏读(红删除线)、`\pse`=停顿(`/`)。
- **RA**：上色对象是**原文**（你照读的那段），含 `\pse` 停顿、`\sr` 读差的词。
- **RS**：上色对象是**原句**，漏/错的词用 `\smiss`（红删除线），读对的 `\sg`。内容分越低删除线越多。
- **DI / RTS**：上色对象是**你说的话（语音识别原文，常很乱）**，原样保留，按词上色。

> 读色技巧：`zoom.py` 放大「AI 语音识别」那一块（详解卷右侧/卡片下方），dpi 用 380~420，逐词记绿/橙/红与删除线。

## 3. 详解卷布局 & 错位配对（坑）

- 详解卷是 **A3**，每页竖排约 **3 张手机截图**（每张一题的「评分弹窗」）。
- 每张卡片：左边**环形总分**（如 `34/90 Total`）+ 右侧分维度（内容/发音/流利度），右边**AI 语音识别**配色文本。
- ⚠️ **错位**：卡片**下方**灰色区里显示的「原句/题干/范文」往往**属于下一题**（截图滚动位置造成）。
  **按内容配对，不要按位置**：RS/RA 的卡片里 AI 识别文本本身就是该题句子，以它为准；DI/RTS 的范文/题干去**题目卷**取更稳。

**读分数**：`zoom.py` 放大环形分 + 右侧分维度。RA 分数在一张「单项/得分/建议」表里（内容 5/5、发音 X/90、流利度 X/90、总分）。

## 4. ⭐ DI 配图（从题目卷裁，做成子文件夹）

1. 在 `口语2.pdf / 口语3.pdf`（列表视图）里找到 DI 题，那里有**完整清晰**的图。
2. 用 `zoom.py` 裁干净（避开「查看原题」「×」等 UI），dpi 300，存到 `build/paper/img/`：
   ```bash
   python3 build/zoom.py 口语2.pdf 2 0.168 0.118 0.625 0.302 300 di_age   # 例：年龄金字塔
   cp build/zoom/di_age.png build/paper/img/di_age.png
   ```
3. 正文 `\includegraphics[width=…\linewidth]{img/di_age.png}`（宽图 0.95、方图 0.5 左右）。
4. **含图片 → 成品做成子文件夹**：`flatten.py … img`（见 §7）。`build/paper/img/` 是工作目录（已 gitignore），真正随成品走的是子文件夹里的 `img/`。

## 5. ⭐ DI / RTS 范文 & ASQ 答案（从题目卷取）

- **DI 范文**：题目卷列表里每道 DI 下有「答案：」一段模型描述 → 放 `modelbox`，并 `\zh` 翻译。
- **RTS 情景 + 范文**：题目卷里有完整**情景题干**和「答案：」**参考答案**（详解卷里题干常被遮/截断）→ 情景放 `promptbox`，范文放 `modelbox`，各加 `\zh` 翻译。
- **ASQ**：题目卷（1B 在 `口语3.pdf` 列表）里**每题都有 题目 + Answer**。**全部收录**（题目 + 参考答案 + 翻译）；你的录音常 0/1 分，可不收你的作答。

## 6. 版面约定（这套排版规范要保持一致）

- **RA / RS**：**第一行=正确原文（+`\zh`翻译）**，**第二行=你的朗读/复述评测（配色 `asrbox`）**，再跟评分行。
- **DI**：原图 → `\ptesub{范文}` modelbox（+翻译）→ `\ptesub{你的作答 AI 语音识别}` asrbox（配色）→ 评分。
- **RTS**：`\ptesub{情景}` promptbox（+翻译）→ `\ptesub{参考答案}` modelbox（+翻译）→ asrbox（配色）→ 评分（恰当性/发音/流利度）。
- **ASQ**：题目 + `\zh` 翻译 + `\textbf{参考答案}`（绿）+ 中文词义，6 题集中列在一个 `notebox` 里。
- 每个题型前用 `\ptetask{中文}{英文}` 做分隔条。
- **每题都要有中文翻译**（`\zh`）。

总分行示例：`{\small 评分：内容 \sfull{5/5}\quad·\quad 发音 \spart{14/90}\quad·\quad 流利度 \szero{10/90}}`
（内容满分 `\sfull` 绿、发音偏低 `\spart` 橙、流利度低 `\szero` 红；总分放 qblock 标题右上 `\textcolor{cAccent}{\bfseries 12}/90`。）

## 9. 易错点清单

- [ ] **没复刻配色**（只写了文字）——必须逐词 `\sg/\sy/\sr/\smiss/\pse`。
- [ ] **去详解卷抠 DI 图**——图被弹窗盖住，要去**题目卷**裁。
- [ ] **错位配对**——卡片下方的句子常是下一题，按内容配对。
- [ ] **漏了范文/ASQ**——DI/RTS 范文、全部 ASQ 都在题目卷，别漏。
- [ ] **忘加翻译**——每题 `\zh`。
- [ ] **题号读错**——环形分旁的 `#1234` 要放大看准（曾把 #577/#567/#576 看混）。
- [ ] **含图却没做成子文件夹** / 没把 `img/` 一起带上。
