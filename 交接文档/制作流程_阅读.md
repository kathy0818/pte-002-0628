## 1. 输入与题型

- **题目卷**：`原卷素材/阅读1.pdf`（含 Q1–Q10）、`原卷素材/阅读2.pdf`（含 Q11–Q16）。A3，每页是网页长截图。
- **⭐ 补充长截图**：`原卷素材/阅读1-5.png`、`阅读5-10.png`、`阅读11-15.png`
  —— 整页**连续滚动**截图（无翻页缝），专门用来补 PDF 在分页处被截断的**文章中段 / 选项 / 题号**（见 §3）。

PTE Core 阅读题型（1B 实际 16 题，按实际为准）：

| 题型 | 1B 数量 | 每题要素 |
|------|------|----------|
| **FIB (R&W)** 读写填空（下拉选词）| 5（Q1–5）| 文章（含填空作答）+ 每空 4 个选项 + **解析** + 评分 |
| **MCM-R** 多选 | 2（Q6–7）| 文章 + 题目 + 选项 A–G + 正确/你的答案 + 评分（**无逐项解析**）|
| **RO** 段落重排 | 2（Q8–9）| 4 个乱序段落 + 正确顺序 + 你的顺序 + 评分 |
| **FIB(R)** 读填空（拖词入空）| 5（Q10–14）| 文章（含填空）+ **词库**（词数多于空格）+ **解析** + 评分 |
| **MCS-R** 单选 | 2（Q15–16）| 文章 + 题目 + 选项 + 正确/你的答案 + 评分（无逐项解析）|

每空 / 每选项：平台用 **✓ 绿（对）/ ✗ 红（错，后跟正确答案）**。复刻见 §2。

## 2. ⭐ 填空 / 选择 答案标注（核心宏）

`ptestyle.sty` 里（需 `\usepackage{pifont}` 出 ✓✗）：

| 宏 | 含义 | 外观 |
|----|------|------|
| `\rok{词}` | 选对 | 绿 ✓ + 词 |
| `\rno{你的}{正确}` | 选错 | 红 ✗ + 删除你的词 + `(答案: 正确)` |

**文章里每个空就地嵌 `\rok/\rno`**，例：
`The \rno{audience}{beholder} ... as well as \rok{physically}.`

- **选项 Choices**：FIB 每空一行（4 词）、FIB(R) 一个共享词库；正确项 `\textbf{加粗}`。
- **MCM/MCS**：选项 A–G 列出，正确项用 `{\color{cFix}\textbf{...}}`（绿粗）；
  答案行 `正确答案 \textcolor{cFix}{\bfseries B, C, E}\qquad 你的答案 {\color{cWrong}\bfseries C, F}`。

## 3. ⭐ 分页截断（阅读最大的坑）

阅读 PDF 是 A3 长截图，**文章 / 选项 / 题号常在两页交界处被截掉一段**——那段内容**根本不在 PDF 里**。

- 1B 实测被截处：**Q3** 文章首句中段、**Q5** 选项①②、**Q6** 文章中段、**Q8** RO 题号。
- **解决：用整页长截图 PNG**（`阅读1-5/5-10/11-15.png`，连续无缝）。用 `build/pngcrop.py` 按**比例**裁：
  ```bash
  python3 build/pngcrop.py 原卷素材/阅读1-5.png <y0> <y1> <名字>   # y0/y1 是 0~1 比例 → build/zoom/<名字>.png
  ```
- **PDF 清晰但有缝，PNG 连续但稍糊 → 交叉用**：主体用 PDF（`zoom.py`），缺的段落用 PNG（`pngcrop.py`）补。
- 长 PNG 很高（上万像素），**题目分布不均**（解析长的题占得多），裁的时候多试几个比例定位。
- 实在补不全：照实标红字 `（原图分页处未截全）`，**别瞎补**（1B 曾这样标，后由考生补全长截图）。

## 4. 解析（仅 FIB / FIB(R) 有）

- **MCM / MCS / RO 没有逐项解析**，只有 `答案`（正确）+ `你的答案`。
- **FIB / FIB(R) 有解析**：每空一条 = **根据原文**（引出带空的原句）+ 为什么选它 + 其他选项词义 + **解题思路**（单句理解 / 固定搭配 / 上下文 / 词性 / 时态 / 主谓一致 / 被动语态…）。
- 平台解析中文字号小、逐字 OCR 困难 → 做**忠实还原**（保留「根据原文 + 思路 + 选项词义」的结构与含义），内容与平台一致即可，不必逐字。

## 5. 翻译（每题都加）

`\zh{中文}` 给**每篇文章 / 每组段落**加整段翻译，放在 `promptbox` 正下方。

## 6. 逐题 LaTeX 结构

**FIB 模板**：

```latex
\begin{qblock}{Q1\quad FIB \#74\ \ Good Looks in Votes}{读写填空 · 评分 \textcolor{cAccent}{\bfseries 2 / 4}}
\ptesub{文章 Passage（含填空作答）}
\begin{promptbox} …文章，空处 \rok{对} / \rno{你的}{正确}… \end{promptbox}
\zh{…整段翻译…}
\ptesub{选项 Choices}
{\small 1.\ a,\ \textbf{b},\ c,\ d\\ 2.\ … }
\ptesub{解析 Explanation}
\begin{notebox}\footnotesize
\textbf{1.\ b} —— 根据原文：…；其他选项 …；\textbf{思路：单句理解}。\\[2pt] …
\end{notebox}
\end{qblock}
```

- **MCM / MCS**：`promptbox` 文章 + `\zh` + `\ptesub{题目}`（`notebox`）+ `\ptesub{选项 Choices}`（A–G，正确项绿粗）+ 答案行。
- **RO**：`promptbox` 内 4 段，每段 `\textbf{1)}…`；`\zh` 翻译；末尾 `正确顺序 \textcolor{cFix}{\bfseries 3 → 2 → 1 → 4}\qquad 你的顺序 …`。
- **FIB(R)**：词库写成一行 `\textbf{正确词}…｜ 干扰词…（多余）`。

---

## 7. 读图技巧（阅读专用）

- 阅读卡片**比口语宽**：全宽约 `x 0.135 ~ 0.885`（口语约 0.30 起）。一开始按窄列裁会切掉左半句。
- **`你的答案: …` 行**一次列出某题所有空的作答 → 配合选项加粗，即可判定每空对错（不必逐空找 ✓/✗）。
- 分数徽章 **`评分详情 X/Y分`** 在每题末（MCM 是「对的-错的」净分，如选 3 对 1 错可能 = 1/2）。
- 题号 `N. 题型 #ID 标题` 在每题开头；ID 放大看准（1B 曾把 #50 看成 #59、#483 看成 #485）。

## 9. 出版本 & 自检清单

阅读无图 → 纯文字版本：

```bash
cd build/paper && xelatex preview_reading.tex && xelatex preview_reading.tex   # 预览核对
python3 build/flatten.py reading 阅读 <编号> <序号>     # → 整理卷/阅读/<序号>_…_阅读_整理卷.tex
cd 整理卷/阅读 && xelatex <序号>_PTE模考<编号>_阅读_整理卷.tex && xelatex … && rm -f *.aux *.log *.out
# 最终版：把该 pdf 复制到 整理卷/ 根并改成干净名 PTE模考<编号>_阅读_整理卷.pdf（见总册 §4）
```

**自检（逐题对原图）**：
- [ ] 文章 / 段落一字不差，空处 `\rok/\rno` 与平台 ✓/✗ 一致，且有 `\zh` 翻译。
- [ ] 选项正确项加粗；MCM/MCS 正确答案绿、你的答案对错配色对。
- [ ] FIB/FIB(R) 每空解析含「根据原文 + 思路」；MCM/MCS/RO 无解析只列答案。
- [ ] 分页截断处：能用长截图补的已补；补不全的标红字说明，未瞎补。
- [ ] 题号 ID、得分与原图核对一致。
