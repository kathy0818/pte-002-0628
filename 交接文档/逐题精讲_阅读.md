- **FIB / FIBD&D：每一空都讲，且每空的 4(或词库多个)选项逐个分析**：✅正确词(为什么对)+ ❌另外几个干扰项**各自为什么错**（含词库里"根本没用上的干扰词"）。能说清干扰项为什么错，才算真会。

## 2. 题型通法

**FIB 读写填空(下拉 4 选 1，只扣选错空) / FIBD&D 拖词填空(共享词库)** —— 三步铁律：
1. **先判词性 + 用 a/an 砍一半**（`an ___`→元音开头；`the+___+impaired`→副词；`be ___ by`→分词；`proper ___ shows`→名词；`These ___ damage`→动词；`stays ___`→形容词）。
2. **形近/义近词逐个翻译再选**（defer/differ、diversity/difference、-fer 家族）。
3. **选完把整句默读一遍**，不通就换。
FIBD&D 额外两条：**词库有多余干扰词→不硬凑**；**同根不同词性(productive/productivity、accounting)按形式选**；**填完通读查"两词放反"**(本考生反复犯 make/use、center/end、availability 错位)。

**MCM-R 多选**：**绝对词(all/never/only/definitely)先怀疑**(原文多是 most/some)；**正确项常是同义替换**(27/35=more than half、structured=organization)；**选全**(多选漏选都扣)。

**MCS-R 单选**：定位句 + 同义替换 + 排除"绝对/歪曲/笼统/与原文相反"；主旨题选**覆盖全文中心**的(排"只讲细节/引子"和"与原文矛盾")。

**RO 段落重排**：① 锁首句(独立、引入话题、**无回指**；人物+背景/问句天然像首句；含 this/these/it/such/the+已知名词/And/So 的不能开头)② 连接词定位(And 收尾、In fact 紧跟被强化句、So now that 开场)③ 代词回指串链 → 通读确认。

## 3. 错因归类库
词性/冠词没用 / 形近义近词混淆 / 选完不回读 / **FIBD&D 两词放反** / 用干扰词硬凑 / 绝对化陷阱 / 没识别同义替换 / MCM 漏选 / RO 首句判断错 / 固定句型不熟(be noted that)。
