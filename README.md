## 1. 这个项目在干什么

把 **APEUni / 猩际 PTE Core 模考**的「答案 & 评分详情」截图（原始不可复制、只能看图）**逐题人工誊写、重新排版**，做成干净、可读、可打印的 **错题与详解整理卷 PDF**。

- 当前这套是 **PTE Core 模考 2B**（总分 50：听力 47 / 阅读 46 / 口语 36 / 写作 71；提交时间 2026-06-28 11:49）。
- 每个版面完整保留每题的：**题干 / 原文、范文、你的作答、逐项评分、彩色批改、AI 中文建议、中文翻译**。
- 错误按平台标注用颜色还原；平台**漏标**的错误用橙色"补"标记额外补上。
- 成品强调"忠于原图"：内容均为人工对照誊录，若与原图有出入以原图为准。

> ⚠️ **本套素材与 1B 不同**：1B 是 PDF 题目卷+PDF 详解卷；**2B 是 docx（内嵌截图）+ png/jpeg 长截图**（清单见 [`交接文档/00…`](交接文档/00_总交接文档·从这里开始.md) §4 与总册 §3）。

## 4. 成品版本规则（重要 —— 必须遵守）

1. **一个版本 = 一对成品**：`<序号>_PTE模考2B_<部分>_整理卷` 的 `.pdf` + `.tex`。
2. **版本化存档按 part 分文件夹**：`整理卷/{写作,口语,阅读,听力}/`（`flatten.py` 已直接输出到这里）。
3. **根目录只放最终成品 PDF**：每 part 最新版另复制一份到 `整理卷/` 根，改干净名 `PTE模考2B_<部分>_整理卷.pdf`。
4. **每个 part 各自从 `000` 起编号**（互不干扰），每更新一版 **+1**（三位数）。
5. **绝不修改、绝不覆盖任何旧版本**——只**新增**下一个序号。
6. **纯文字版本**（写作/阅读/听力）：`.tex` 单文件自包含，放 `整理卷/<部分>/`。
   **含图片版本**（口语 DI 配图）：做成**子文件夹** `整理卷/口语/<序号>_…/`（`.tex` + `img/` + `.pdf`）。
7. **原始素材**统一在 `原卷素材/`（只读）；**交接文档**统一在 `交接文档/`。

## 5. 环境与编译

**引擎：XeLaTeX**（中英混排，需 CJK 字体）。需要的字体：`Noto Sans CJK SC`、`Noto Serif CJK SC`、`TeX Gyre Heros`。

```bash
sudo apt-get install -y texlive-xetex texlive-latex-extra texlive-latex-recommended \
    texlive-plain-generic texlive-lang-chinese fonts-noto-cjk fonts-texgyre poppler-utils
```

**编译某一版成品**（单文件，自包含，跑两遍）：

```bash
cd 整理卷/写作
xelatex 000_PTE模考2B_写作_整理卷.tex && xelatex 000_PTE模考2B_写作_整理卷.tex
```

**在工作区预览/继续排版**：

```bash
cd build/paper
xelatex preview_writing.tex && xelatex preview_writing.tex
```

## 6. 批改标注图例（来自 `ptestyle.sty`）

| 宏 | 含义 | 颜色 |
|----|------|------|
| `\fix{原}{正}` | 修改 / 拼写 / 缺空格 | 红删除线 + 绿改正 |
| `\del{词}` | 应删除 | 蓝删除线 + "删" |
| `\ins{词}` | 应插入 | 紫色 + "∧插" |
| `\add{原}{正}` | **平台漏标**、本卷补标 | 橙色波浪线 + "补" |
| `\rok{词}` / `\rno{你的}{正确}` | 阅读：选对 ✓ / 选错 ✗ | 绿 / 红 |
| `\sfull/\spart/\szero` | 评分：满分 / 部分 / 零分 | 绿 / 橙 / 红 |
| `\zh{中文}` | 中文翻译行 | 蓝"译" + 灰译文 |

颜色主色为平台青绿 `cAccent = #16A085`；四项分数条形图配色见 `cover.tex`。
